  1. iPhone 설정 앱 → Safari → 아래로 스크롤 → Camera → Ask 또는 Allow
  2. 또는 Safari URL 바의 AA 버튼 → Website Settings → Camera → Allow
  3. 또는 Settings → Safari → Advanced → Website Data → github.io 검색 후 삭제
  4. Private Browsing(시크릿) 탭이면 일반 탭으로 전환
  5. 위 조치 후 페이지 새로고침
